# Rusky
To start off the bot use the command:-
```
Cargo run
```

# Features
```
•Chatbot
•Music With Lavalink-rs 
•Songbird Implenementation
•Command handler
•Sql Working database
•More commands(see in src)
```

# Requirements
```
• Rust Drivers
• Minimum of Rust 1.51.
```
